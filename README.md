# Simple Slavery (RimWorld 1.6 Update)

---

## 📌 Description (ENG)
This is an updated version of Thirite’s **Simple Slavery**, reworked to be compatible with RimWorld 1.6 and the official DLC slavery system.  
The original standalone slavery mechanics have been removed (since DLC covers this), but the **Slave Collar** has been retained, a new **Remote Collar** feature has been added, and several tweaks have been implemented to improve and expand DLC slave mechanics.

### 🔧 Features / Changes
- Full support for RimWorld 1.6 and DLC
- Removed original custom slavery system (DLC system is used instead)
- **Slave Collar** kept + **Remote Collar** feature added
- Various DLC slavery tweaks:  
  (slave mood effects, rebellion cycle, work speed, status assignment, etc.)

### 📥 Installation
- Download from GitHub Releases or Steam Workshop and place in your `Mods` folder.
- Requires RimWorld 1.6 or higher.
- Requires [Harmony](https://steamcommunity.com/sharedfiles/filedetails/?id=2009463077).
- No other dependencies.

## 📌 소개 (KOR)
이 모드는 Thirite의 **Simple Slavery** 모드를 기반으로 RimWorld 1.6 및 DLC의 노예 시스템과 호환되도록 업데이트한 버전입니다.  
기존 모드의 독자적인 노예 시스템은 DLC에 통합되었으므로 제거되었으며, 노예 목걸이 및 **원격 제어 기능**, 그리고 DLC 노예 시스템과 관련된 다양한 트윅이 추가되었습니다.

### 🔧 주요 변경점
- RimWorld 1.6 지원 및 최신 DLC 호환
- 기존 독자 노예 시스템 제거 (DLC 기능 사용)
- **Slave Collar** 유지 + **Remote Collar** 기능 추가
- DLC 노예 시스템 관련 다양한 편의/밸런스 조정  
  (노예 무드 변화, 반란 주기, 작업 속도, 지위 할당 등)

### 📥 설치
- GitHub Release 또는 Steam Workshop에서 다운로드 후 `Mods` 폴더에 넣으세요.
- RimWorld 1.6 이상 필요.
- [Harmony](https://steamcommunity.com/sharedfiles/filedetails/?id=2009463077) 필수.
- 추가 모드 의존성 없음.
